# SeeEDU

## Goal
A crowdfunding project with pal feature

## Framework
Laravel 5.7.21

## Requirement
- Composer
- PHP >= 7.0.0
- OpenSSL PHP Extension
- PDO PHP Extension
- Mbstring PHP Extension
- Tokenizer PHP Extension
- XML PHP Extension

## Run

1. Install Dependency
```shell
composer install
```

2. Migrate and Seeding
```shell
php artisan migrate:refresh --seed
```

3. Run Server
```shell
php artisan serve
```
